vue实战之（二十三）使用`axios`发送`get`和`post`请求

2019-03-23 00:19:05
 / 本文总阅读量 次
#$axios

前言

发送ajax请求应该说是学习前端的基本功，今天我们就来总结一下vue框架中使用axios发送get和post请求。

get请求

  • 基本格式

需要注意的是参数部分必须这么写{params: {key: value}},不需要传参的话：{}，也可以省略不写，否则无法传参

post请求

var formData=new FormData();
formData.append('user',123456);
formData.append('pass',12345678);

axios.post("/notice",formData)
     .then((res) => {return res})
     .catch((err) => {return err})

需要注意的是：在使用data中定义的参数接收返回数据的时候，需要在方法中重新定义this指向,var _this = this
否则会出现接收不到的情况。

data () {
    return {
      tableData: [],
      }
}

methods: {
    getData () {
      var _this = this// 这个非常重要，没有话接收不到参数
      this.$axios.get('/getUsers')
        .then(function (response) {
          if (response.data.success) {
            _this.tableData = response.data.data
            console.log(_this.tableData)
            _this.$toast('数据加载完成', true, 3000)
          }
          var errorMsg = response.data.errorMes
          _this.$toast(errorMsg, true, 3000)
        }).catch(function (error) {
          _this.$toast(error, false, 3000)
        })
    }
}

全局引入axios

在项目中安装axios,使用命令：npm install axios --save

在main.js中添加配置

var axios = require('axios')
// 设置请求格式
axios.defaults.headers.post['Content-Type'] = 'application/x-www-form-urlencoded'
// 全局请求路径
axios.defaults.baseURL = 'http://localhost:8088/demo'
// 将API方法绑定到全局
Vue.prototype.$axios = axios

在页面中直接使用this.$axios.get或this.$axios.post

局部引入axios

import axios from 'axios';

在页面中直接使用axios.get或axios.post
